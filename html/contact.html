Si vous avez des questions, suggestions ou besoin d'aide, vous pouvez utiliser le formulaire ci-dessous pour nous contacter.

FAQ (Foire Aux Questions)

Pour créer un compte, cliquez sur "S'inscrire" en haut de la page, puis remplissez le formulaire avec vos informations personnelles.

Pour ajouter une annonce aux favoris, cliquez sur le bouton "Ajouter aux favoris" présent sur la page de l'annonce.

Pour contacter un vendeur, cliquez sur le bouton "Contacter le vendeur" sur la page de l'annonce. Vous pourrez envoyer un message directement.
